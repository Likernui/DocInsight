# DocInsight

Десктопное приложение для извлечения сущностей и семантического поиска по корпусу документов.

## Назначение

Автоматическое извлечение структурированных данных (ФИО, научные руководители, источники, технологии и др.) из документов:
- Отчёты (DOCX)
- Документы (PDF)
- Сканы (PNG, JPG)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Сборка в EXE

```bash
pyinstaller --onefile --windowed main.py
```

## Архитектура

```
Загрузка документов → Извлечение текста → OCR (для сканов)
                                              ↓
Предобработка → Разбиение на фрагменты → Создание эмбеддингов → FAISS индекс
                                              ↓
Поиск по запросу → NER + Regex → Агрегация → Генерация DOCX
```

## Модули

- `text_extractor.py` — извлечение текста из DOCX, PDF, изображений
- `preprocessor.py` — очистка и разбиение текста на фрагменты
- `indexer.py` — создание векторного индекса
- `semantic_search.py` — семантический поиск
- `entity_extractor.py` — извлечение сущностей (NER + regex)
- `aggregator.py` — агрегация результатов
- `result_generator.py` — генерация итогового DOCX
- `ui/main_window.py` — PyQt интерфейс
